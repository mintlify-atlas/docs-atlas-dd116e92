CAD Settings allow you to customize various aspects of your SnailyCAD instance. These settings are managed through the admin panel and stored in the database.

<Note>
  You must have the **ManageCADSettings** permission to access and modify these settings.
</Note>

## General Settings

Access general CAD settings from **Admin** > **Manage** > **CAD Settings**.

### Basic Information

<ParamField path="name" type="string">
  The name of your CAD. This appears in the browser title and throughout the application.
</ParamField>

<ParamField path="areaOfPlay" type="string">
  The current area of play (AOP). This is displayed to all users and can be updated in real-time.
  
  Example: `"Los Santos"`, `"San Andreas"`, `"Liberty City"`
</ParamField>

<ParamField path="timeZone" type="string">
  The timezone for your CAD. Affects timestamp displays throughout the application.
  
  Example: `"America/New_York"`, `"Europe/London"`, `"UTC"`
</ParamField>

### Whitelisting

<ParamField path="whitelisted" type="boolean" default={false}>
  When enabled, only users with the whitelisted role can register and access the CAD.
</ParamField>

<ParamField path="businessWhitelisted" type="boolean" default={false}>
  When enabled, business creation requires approval from an administrator.
</ParamField>

<ParamField path="towWhitelisted" type="boolean" default={false}>
  When enabled, only whitelisted users can create tow characters.
</ParamField>

<ParamField path="taxiWhitelisted" type="boolean" default={false}>
  When enabled, only whitelisted users can create taxi characters.
</ParamField>

### Registration

<ParamField path="registrationCode" type="string">
  An optional code required for new user registration. Leave empty to allow open registration.
  
  <Warning>
    The registration code is visible to administrators only.
  </Warning>
</ParamField>

### Branding

<ParamField path="logoId" type="string">
  The ID of the uploaded logo image. Upload images through the CAD Settings Images section.
</ParamField>

<ParamField path="cadOGDescription" type="string">
  The Open Graph description for your CAD. This appears when sharing links on social media.
</ParamField>

<ParamField path="authScreenBgImageId" type="string">
  The ID of the background image for authentication screens (login/register).
</ParamField>

<ParamField path="authScreenHeaderImageId" type="string">
  The ID of the header image for authentication screens.
</ParamField>

<ParamField path="roleplayEnabled" type="boolean" default={true}>

<ParamField path="maxCitizensPerUser" type="number" default={null}>

### Templates

<ParamField path="callsignTemplate" type="string">
  Template for generating unit callsigns.
  
  Available variables:
  - `{department}` - Department identifier
  - `{division}` - Division identifier
  - `{officerNumber}` - Officer number
  
  Example: `"{department}-{division}-{officerNumber}"`
</ParamField>

<ParamField path="pairedUnitTemplate" type="string">
  Template for generating paired unit callsigns.
  
  Example: `"{callsign1}+{callsign2}"`
</ParamField>

<ParamField path="caseNumberTemplate" type="string">
  Template for generating case numbers.
  
  Example: `"#{year}-{caseNumber}"`
</ParamField>

## Features

Toggle various features on or off for your CAD instance.

Access from **Admin** > **Manage** > **CAD Settings** > **Features**.

Features can be enabled or disabled individually. Some features may have additional configuration options.

## API Token

Generate and manage API tokens for programmatic access to your CAD.

<ParamField path="enabled" type="boolean" default={false}>
  Enable or disable the global API token.
</ParamField>

When enabled, an API token is generated with the prefix `sng_` followed by 56 random characters.

<Warning>
  Keep your API token secure! Anyone with this token has full API access to your CAD.
</Warning>

### Managing API Tokens

- **Enable**: Creates a new API token
- **Disable**: Removes the current API token
- **Regenerate**: Creates a new token, invalidating the previous one

## Default Permissions

Set default permissions for new users.

<ParamField path="defaultPermissions" type="array">
  Array of permission names that are automatically granted to new users.
  
  See the [Permissions documentation](/configuration/permissions) for available permissions.
</ParamField>

## Blacklisted Words

Manage words that are not allowed in citizen names, vehicle plates, and other user inputs.

<ParamField path="word" type="string">
  A word to blacklist. All comparisons are case-insensitive.
</ParamField>

Blacklisted words can be:
- Added individually through the admin panel
- Imported in bulk via CSV/JSON file
- Deleted individually

## Live Map Settings

Configure live map integration for real-time unit tracking.

See the [Live Map Integration guide](/integrations/live-map) for detailed setup instructions.

<ParamField path="liveMapURLs" type="array">
  Array of live map URLs that can display unit positions.
</ParamField>

## Audit Logs

All changes to CAD settings are automatically logged in the audit log, including:

- CAD settings updates
- Feature toggles
- Miscellaneous settings changes
- Default permissions changes
- API token operations
- Discord role configuration

View audit logs from **Admin** > **Manage** > **CAD Settings** > **Audit Logs**.
