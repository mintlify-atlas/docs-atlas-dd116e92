## Overview

The Admin API provides endpoints for managing CAD settings, users, values, and administrative operations.

<Warning>
All admin endpoints require appropriate administrative permissions. Access is restricted based on user rank and specific permission grants.
</Warning>

---

## Get Dashboard Statistics

<CodeGroup>
```bash GET /api/admin
curl -X GET https://your-cad.com/api/admin \
  -H "Cookie: snaily-cad-access-token=<token>"
```

</CodeGroup>

Get administrative dashboard statistics and metrics.

</ResponseField>

<ResponseField name="pendingUsers" type="number">

<ResponseField name="bannedUsers" type="number">
  Number of banned users
</ResponseField>

<ResponseField name="createdCitizens" type="number">
  Total citizens created
</ResponseField>

<ResponseField name="arrestCitizens" type="number">
  Citizens with arrest records
</ResponseField>

<ResponseField name="deadCitizens" type="number">
  Citizens marked as deceased
</ResponseField>

<ResponseField name="officerCount" type="number">
  Total officers
</ResponseField>

<ResponseField name="onDutyOfficers" type="number">
  Currently on-duty officers
</ResponseField>

<ResponseField name="suspendedOfficers" type="number">
  Suspended officers
</ResponseField>

<ResponseField name="emsDeputiesCount" type="number">
  Total EMS-FD deputies
</ResponseField>

<ResponseField name="onDutyEmsDeputies" type="number">
  Currently on-duty deputies

<ResponseField name="vehicles" type="number">
  Total registered vehicles
</ResponseField>

<ResponseField name="impoundedVehicles" type="number">
  Impounded vehicles
</ResponseField>

## User Management

### Get Users

<CodeGroup>
```bash GET /api/admin/manage/users
curl -X GET "https://your-cad.com/api/admin/manage/users?skip=0" \
  -H "Cookie: snaily-cad-access-token=<token>"
```
</CodeGroup>

Get all users with pagination and filtering.

#### Query Parameters

<ParamField query="skip" type="number" default="0">
  Pagination offset
</ParamField>

<ParamField query="includeAll" type="boolean" default="false">
  Include all users
</ParamField>

<ParamField query="query" type="string">
  Search by username
</ParamField>

### Update User

<CodeGroup>
```bash PUT /api/admin/manage/users/:id
curl -X PUT https://your-cad.com/api/admin/manage/users/{userId} \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "rank": "ADMIN",
    "permissions": ["Leo", "Dispatch"],
    "whitelistStatus": "ACCEPTED"
  }'
```
</CodeGroup>

Update a user's rank, permissions, or whitelist status.

#### Request Body

<ParamField body="rank" type="string">
  User rank: `OWNER`, `ADMIN`, `USER`
</ParamField>

<ParamField body="permissions" type="array">
  Array of permission strings
</ParamField>

<ParamField body="whitelistStatus" type="string">
  Whitelist status: `ACCEPTED`, `PENDING`, `DECLINED`
</ParamField>

### Ban/Unban User

<CodeGroup>
```bash POST /api/admin/manage/users/:id/ban
curl -X POST https://your-cad.com/api/admin/manage/users/{userId}/ban \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "banned": true,
    "banReason": "Violation of terms"
  }'
```
</CodeGroup>

Ban or unban a user.

#### Request Body

<ParamField body="banned" type="boolean" required>
  Ban (true) or unban (false)
</ParamField>

<ParamField body="banReason" type="string">
  Reason for ban (required when banning)
</ParamField>

## Values Management

### Get Values

<CodeGroup>
```bash GET /api/admin/values/:path
curl -X GET https://your-cad.com/api/admin/values/penal_code \
  -H "Cookie: snaily-cad-access-token=<token>"
```
</CodeGroup>

Get values for a specific category.

#### Path Parameters

<ParamField path="path" type="string" required>
  Value category: `penal_code`, `vehicle`, `weapon`, `gender`, `ethnicity`, `license`, etc.
</ParamField>

### Create Value

<CodeGroup>
```bash POST /api/admin/values/:path
curl -X POST https://your-cad.com/api/admin/values/penal_code \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "value": "Assault",
    "type": "FELONY",
    "jailTime": {"min": 10, "max": 20},
    "fine": {"min": 500, "max": 1000}
  }'
```
</CodeGroup>

Create a new value.

### Update Value

<CodeGroup>
```bash PUT /api/admin/values/:path/:id
curl -X PUT https://your-cad.com/api/admin/values/penal_code/{valueId} \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "value": "Aggravated Assault"
  }'
```
</CodeGroup>

Update an existing value.

### Delete Value

<CodeGroup>
```bash DELETE /api/admin/values/:path/:id
curl -X DELETE https://your-cad.com/api/admin/values/penal_code/{valueId} \
  -H "Cookie: snaily-cad-access-token=<token>"
```
</CodeGroup>

Delete a value.
