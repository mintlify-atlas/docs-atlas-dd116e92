## Overview

The Citizens API provides endpoints for creating and managing citizens, their vehicles, weapons, medical records, and licenses.

<Info>
All citizen endpoints require authentication. Some endpoints may be restricted based on CAD features and user permissions.
</Info>

---

## Get Citizens

<CodeGroup>
```bash GET /api/citizen
curl -X GET "https://your-cad.com/api/citizen?query=John&skip=0" \
  -H "Cookie: snaily-cad-access-token=<token>"
```
</CodeGroup>

Get all citizens belonging to the authenticated user.

### Query Parameters

<ParamField query="query" type="string">
  Search by name, surname, SSN, or phone number
</ParamField>

<ParamField query="skip" type="number" default="0">
  Number of records to skip for pagination
</ParamField>

### Response

<ResponseField name="citizens" type="array">
  Array of citizen objects
</ResponseField>

<ResponseField name="totalCount" type="number">
  Total number of citizens matching the query
</ResponseField>

## Create Citizen

<CodeGroup>
```bash POST /api/citizen
curl -X POST https://your-cad.com/api/citizen \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "John",
    "surname": "Doe",
    "dateOfBirth": "1990-01-15",
    "gender": "male-gender-id",
    "ethnicity": "ethnicity-id",
    "weight": "180",
    "height": "72",
    "hairColor": "brown",
    "eyeColor": "blue",
    "address": "123 Main St",
    "phoneNumber": "555-0123"
  }'
```
</CodeGroup>

Create a new citizen.

### Request Body

<ParamField body="name" type="string" required>
  First name
</ParamField>

<ParamField body="surname" type="string" required>
  Last name
</ParamField>

<ParamField body="dateOfBirth" type="string" required>
  Date of birth (ISO 8601 format)
</ParamField>

<ParamField body="gender" type="string" required>
  Gender value ID
</ParamField>

<ParamField body="ethnicity" type="string" required>
  Ethnicity value ID
</ParamField>

<ParamField body="weight" type="string">
  Weight
</ParamField>

<ParamField body="height" type="string">
  Height
</ParamField>

<ParamField body="hairColor" type="string">
  Hair color
</ParamField>

<ParamField body="eyeColor" type="string">
  Eye color
</ParamField>

<ParamField body="address" type="string">
  Residential address
</ParamField>

<ParamField body="phoneNumber" type="string">
  Phone number
</ParamField>

<ParamField body="socialSecurityNumber" type="string">
  Social security number (auto-generated if not provided)
</ParamField>

### Response

Returns the created citizen object with all fields.

### Error Responses

- `400` - Invalid data, blacklisted name, or duplicate name (if not allowed)
- `400` - Maximum citizens per user reached

## Delete Citizen

<CodeGroup>
```bash DELETE /api/citizen/:id
curl -X DELETE https://your-cad.com/api/citizen/{citizenId} \
  -H "Cookie: snaily-cad-access-token=<token>"
```
</CodeGroup>

Delete a citizen permanently.

### Path Parameters

<ParamField path="id" type="string" required>
  Citizen ID
</ParamField>

### Response

Returns `true` on success.

### Error Responses

- `403` - Only admins can delete citizens (if feature is disabled)
- `404` - Citizen not found

## Vehicles

### Register Vehicle

```bash POST /api/vehicles
curl -X POST https://your-cad.com/api/vehicles \
  -H "Cookie: snaily-cad-access-token=<token>" \
  -H "Content-Type: application/json" \
  -d '{
```
