## What is SnailyCAD?

SnailyCAD is a comprehensive, open source Computer Aided Dispatch (CAD) system designed for FiveM roleplay communities. Built with TypeScript and modern web technologies, it provides a professional-grade platform for police roleplaying and emergency services dispatch operations.

Whether you're running a small community or a large-scale roleplay server, SnailyCAD offers the tools you need to manage law enforcement, emergency medical services, fire departments, and civilian operations.

## Built With Modern Technology

SnailyCAD is built using a modern tech stack to ensure performance, reliability, and ease of development:

- **Frontend**: Next.js, React, TypeScript
- **Backend**: Node.js (v18.16.0+), Prisma ORM
- **Database**: PostgreSQL
- **Package Manager**: pnpm
- **Architecture**: Monorepo with Turbo

<Note>
  SnailyCAD requires Node.js version 18.16.0 or higher to run properly.
</Note>
